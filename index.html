Online Remote Classroom API Document

ORC Api are written using

  • Node Js
  • Express
  • MongoDB
  • JWT

To access to the API on the front end level, AXIOS module is required!

Each access are defined and control by different gates called as Route Each route are available for different purposes. And should be used to retrieve a specific data as required.

Description : Use to sign up one user.

Route :

http://localhost:3000/auth/signUp

AXIOS Parameters : Email, Password, Name 

                                
    axios.post(`http://localhost:3000/auth/signUp`, {
            email,
            pwd,
            name
        })

Return Data : On each successful request, data will be returned as following 

            
    {
        classOwnerShip : [],
        classParticipated : [],
        email : email ,
        name : name
    }

Failed Data : On each insuccessful request, data will be returned as following 

            
    {
        message : err.message
    }
    // Error Message include : "Email is already registered!" , Validation Error Message

Description : Use to verify and logged in one user.

Route :

http://localhost:3000/auth/login

AXIOS Parameters : Email, Password 

                                
    axios.post(`http://localhost:3000/auth/login`, {
        email,
        pwd
    })

Return Data : On each successful request, data will be returned as following 

            
    {
        message : Login Success,
        token : token
    }
    // Token is very important in session and credentials management, should be handled and assigned properly to use authorized requests.

Failed Data : On each insuccessful request, data will be returned as following 

            
    {
        message : err.message
    }
    // Error Message include : "Password entered is incorrect" , "Email does not exist"

Route :

http://localhost:3000/users/user

AXIOS Parameters : Token ( in headers ) 

                                
    axios.get(`http://localhost:3000/users/user`,{ params:{}, headers: { 'auth-token': token } })

Description : Create a new class which will be owned by the current user. Parameter including password can be passed as an empty string in case isPrivate is false.

Route :

http://localhost:3000/users/createClass

AXIOS Parameters : Token ( in headers ), classroomName, isPrivate, password 

                                
    axios.post(`http://localhost:3000/users/createClass`,{
                    classroomName,
                    isPrivate,
                    password
                },{ params:{}, headers: { 'auth-token': token } })

Return Data : On each successful request, data will be returned as following 

    
    {
        classroomName: classroomName ,
        code: code,
        isPrivate : false,
        password : "",
        currentlyStreaming: "",
        members: [],
        owner: email
    }

Failed Data : On each insuccessful request, data will be returned as following 

            
    {
        message : err.message
    }

      code: code,
      isPrivate : false,
      password : "",
      currentlyStreaming: "",
      members: [],

  }

Failed Data : On each insuccessful request, data will be returned as following

Description : Start the stream in the current classroom.

Route :

http://localhost:3000/users/startstream

AXIOS Parameters : Token ( in headers ), code ( classroom's code ), streamTitle 

                              
  axios.post(`http://localhost:3000/users/startstream`,{
                code,
                streamTitle
              },{ params:{}, headers: { 'auth-token': token } })

Return Data : On each successful request, data will be returned as following 

  
  {
    code : code,
    streamCode : code,
    streamTitle : streamTitle,
    owner : owner-email,
    date : Date-Created
  }

Failed Data : On each insuccessful request, data will be returned as following 

          
  {
      message : err.message
  }
